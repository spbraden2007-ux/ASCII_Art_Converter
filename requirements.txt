# ASCII Art Converter - Requirements
# ====================================
# Minimal dependencies for image-to-ASCII conversion

# Image Processing
Pillow>=8.0.0       # Image loading, grayscale conversion, resizing, pixel extraction

# Python version requirement:
# Python >= 3.6

# Installation:
# pip install -r requirements.txt

# Alternative installation:
# pip install Pillow
# conda install pillow
# Ubuntu: sudo apt-get install python3-pil
# macOS: brew install pillow

# Notes:
# - Pillow (PIL fork) is the only external dependency
# - Built-in libraries used: os (file I/O)
# - Supports: .jpg, .jpeg, .png, .gif, .webp image formats
